dds.DepositionTarget

class dds.DepositionTarget(position, normal=(0.0, 0.0, 1.0))[source]

Bases: object

A top-referenced position and deposition normal.

Parameters:

  • position (dds.primitives.Point3D) – World-space top position of the deposited bead, usually the nozzle target.

  • normal (dds.primitives.Vector3D) – Deposition normal. The vector is normalized during construction.

Notes

Targets are top-referenced, not center-referenced: bead geometry extends opposite the normal by BeadProfile.height.
