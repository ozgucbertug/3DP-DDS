class dds.analysis.StratumFieldSet(domain, threshold, stratum_ids, implicit_fields, occupancy_fields, label_field)[source]

Bases: object

Implicit and occupancy fields partitioned by deposit order.

Parameters:

  • domain (dds.domain.Domain) – Domain shared by all stored fields.

  • threshold (float) – Occupancy threshold used to construct the strata.

  • stratum_ids (tuple[int, ...]) – One-based deposition-order identifiers included in this result.

  • implicit_fields (tuple[ndarray[Any, dtype[float64]], ...]) – Per-stratum fields with shape domain.grid_shape.

  • occupancy_fields (tuple[ndarray[Any, dtype[bool_]], ...]) – Per-stratum fields with shape domain.grid_shape.

  • label_field (numpy.ndarray[Any, numpy.dtype[numpy.float64]]) – Dense field containing the stratum identifier at each occupied voxel.
