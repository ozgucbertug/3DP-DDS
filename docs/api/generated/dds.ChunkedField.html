dds.ChunkedField

class dds.ChunkedField(domain, chunk_shape=(32, 32, 32), include_coverage=False)[source]

Bases: object

Sparse field backed by fixed-size dense chunks.

Every chunk stores the implicit field. Coverage is allocated only when requested. Empty chunks are never stored.

Parameters:

  • domain (dds.domain.Domain) – Simulation domain for the chunked storage.

  • chunk_shape (tuple[int, int, int]) – Dense chunk dimensions in (x, y, z) index order.

  • include_coverage (bool) – Allocate additive coverage chunks in addition to implicit chunks.
