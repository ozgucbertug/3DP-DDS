dds.Simulator

class dds.Simulator(domain, deposits=None)[source]

Bases: object

Stateful deposit collection with incrementally updated dense caches.

Parameters:

  • domain (Domain) – Simulation domain used for all deposits and result snapshots.

  • deposits (Optional[Union[Iterable[DepositInput], DepositInput]]) – Optional initial deposit or iterable of deposits.

Notes

Simulator is the mutable construction API. Calls to result() return immutable snapshots; later calls to add_deposit(), add_deposits(), or clear_deposits() do not mutate snapshots that have already been returned.

property deposits: tuple[PointDeposit | LineDeposit | PolylineDeposit, ...]
