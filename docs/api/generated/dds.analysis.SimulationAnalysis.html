dds.analysis.SimulationAnalysis

class dds.analysis.SimulationAnalysis(domain, implicit_field, deposits, default_threshold=0.5, *, _copy_implicit_field=True)[source]

Bases: object

Cached queries derived lazily from an immutable simulation field.

Parameters:

  • domain (dds.domain.Domain) – Simulation domain that defines bounds, voxel size, and grid shape.

  • implicit_field (npt.NDArray[np.float64]) – Nonnegative union-like deposition field with shape domain.grid_shape.

  • deposits (tuple[dds.primitives.PointDeposit | dds.primitives.LineDeposit | dds.primitives.PolylineDeposit, ...]) – Deposits that produced the field, in simulation order.

  • default_threshold (float) – Threshold used when query methods are called without an explicit threshold.

  • _copy_implicit_field (bool)

Notes

The stored implicit field is not a signed-distance field. It is clipped to [0, 1] and uses the threshold, commonly 0.5, as the nominal fabricated surface. Use surface_sdf() or signed_distance_at() for metric distances.
