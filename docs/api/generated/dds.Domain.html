dds.Domain

class dds.Domain(min_corner, max_corner, voxel_size, grid_shape, length_unit='mm')[source]

Bases: object

Axis-aligned simulation workspace sampled on a dense 3D grid.

Parameters:

  • min_corner (tuple[float, float, float]) – Lower inclusive and upper exclusive world-space bounds.

  • max_corner (tuple[float, float, float]) – Lower inclusive and upper exclusive world-space bounds.

  • voxel_size (tuple[float, float, float]) – Positive voxel spacing in (x, y, z) order.

  • grid_shape (tuple[int, int, int]) – Number of voxels in (x, y, z) order.

  • length_unit (Literal['mm', 'm']) – Informational unit label, either "mm" or "m".

Notes

Dense arrays in 3DP-DDS use (x, y, z) index order and NumPy indexing="ij" conventions. Domain records units but does not convert between them.

Create a domain from scalar bounds and voxel size.
